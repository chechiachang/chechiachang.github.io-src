дёҠйқўи¬ӣи§Ј Terraform зҡ„еҹәжң¬ж“ҚдҪңжөҒзЁӢпјҢжҸҗдҫӣзҜ„жң¬еҺҹе§ӢзўјпјҢд»ҘеҸҠдёҖжӯҘдёҖжӯҘе°Һе…Ҙзҡ„и©ізҙ°жӯҘй©ҹгҖӮеҗ„дҪҚжҮүи©ІйғҪеҸҜд»Ҙдҫқз…§дёҠйқўе№ҫзҜҮзҡ„иӘӘжҳҺпјҢй–Ӣе§Ӣеҝ«жЁӮзҡ„дҪҝз”Ё Terraform дәҶгҖӮ

иҖҢз•¶дҪҝз”Ё Terraform зҡ„иҰҸжЁЎи¶ҠдҫҶи¶ҠеӨ§пјҢз®ЎзҗҶзҡ„иіҮж–ҷи¶ҠдҫҶи¶ҠеӨҡжҷӮпјҢй–Ӣе§ӢжңғеҮәзҸҫдёҖдәӣе•ҸйЎҢпјҢдҫӢеҰӮйҮҚиӨҮзҡ„ terraform code и¶ҠдҫҶи¶ҠеӨҡпјҢеҚ”еҗҢе·ҘдҪң review дёҚеӨӘе®№жҳ“пјҢstate зҡ„е…§е®№з®ЎзҗҶиҲҮйҺ–з®ЎзҗҶпјҢзӯүзӯүгҖӮйҖҷдәӣе•ҸйЎҢеҸҜд»ҘйҖҸйҒҺдёҖдәӣе·ҘдҪңжөҒзЁӢзҡ„ж”№йҖІпјҢжҲ–жҳҜе°Һе…Ҙж–°зҡ„е°Ҹе·Ҙе…·пјҢдҫҶж”№е–„е·ҘдҪңж•ҲзҺҮгҖӮ

жҺҘдёӢдҫҶзӯҶиҖ…жҺЁи–Ұе№ҫеҖӢеҝғеҫ—иҲҮе·Ҙе…·пјҢеёҢжңӣиғҪжҸҗеҚҮдҪҝз”Ё Terraform зҡ„ж•ҲзҺҮиҲҮз”ўеҖј

е•ҸйЎҢиҲҮйңҖжұӮ#

з•¶еңҳйҡҠе·Із¶“й–Ӣе§ӢеӨ§иҰҸжЁЎдҪҝз”Ё terraformпјҢtf жӘ”жЎҲи¶ҠдҫҶи¶ҠеӨҡгҖӮжҲ‘еҖ‘зӮәдәҶеўһеҠ зЁӢејҸзўјзҡ„йҮҚиӨҮеҲ©з”ЁжҖ§пјҢжңғдҪҝз”Ё terraform module е°Үеёёз”Ёзҡ„ tf жӘ”жЎҲе°ҒиЈқгҖӮ

  • йҡЁи‘—е°Һе…Ҙзҡ„иҰҸжЁЎи¶ҠдҫҶи¶ҠеӨ§пјҢйҖҷдәӣ module жңғи¶ҠдҫҶи¶ҠеӨҡпјҢиҖҢдё”дҪҝз”ЁйҖҷдәӣ module зҡ„ project д№ҹеўһеҠ 
  • йҡЁи‘—з®ЎзҗҶзҡ„ infrastructure и¶ҠдҫҶи¶ҠиӨҮйӣңпјҢзӮәдәҶжҸҸиҝ°йҖҷдәӣжң¬дҫҶе°ұеҫҲиӨҮйӣңзҡ„ infrastructureпјҢmodule дёҚеҸӘи¶ҠдҫҶи¶ҠеӨҡпјҢйӮ„жңғеҮәзҸҫ module еј•з”Ёе…¶д»– moduleпјҢеӨ§ module дҪҝз”Ёе°Ҹ module зҡ„ nested module
    • йҖҷй»һеңЁиӨҮйӣңзҡ„ provider дёӯеёёеҮәзҸҫпјҢдҫӢеҰӮ
      • дҪҝз”Ё terraform жҸҸиҝ°з¶Іи·Ҝжһ¶ж§Ӣ
      • жҸҸиҝ°иӨҮйӣңзҡ„ IAM & Role
      • terraform vault-provider
  • з®ЎзҗҶзҡ„з’°еўғи¶ҠдҫҶи¶ҠеӨҡпјҢдҫӢеҰӮ dev, staging, prod,…пјҢжҜҸеҖӢз’°еўғйғҪйңҖиҰҒзҚЁз«Ӣзҡ„е·ҘдҪңз©әй–“пјҢйҖ жҲҗеӨ§йҮҸйҮҚиӨҮзҡ„ tf code

дҪҝз”ЁдёҖж®өжҷӮй–“ terraform пјҢеҫҲеҝ«е°ұжңғзҷјзҸҫзҡ„з¬¬дёҖеҖӢе•ҸйЎҢжҳҜпјҡдёҚи«–жҖҺйәјеҫһ tf жӘ”жЎҲдёӯжҸҗеҸ–йҮҚиӨҮйғЁеҲҶпјҢеҒҡжҲҗ moduleпјҢйӮ„жҳҜжңүеҫҲеӨҡйғЁеҲҶзҡ„ tf code жҳҜе®Ңе…ЁйҮҚиӨҮзҡ„пјҢдҫӢеҰӮпјҡ

  • жҜҸеҖӢ module жңғе®ҡзҫ©зҡ„ variable (argument)пјҢдҪҝз”ЁйҖҷеҖӢ module зҡ„дёҠеұӨжңғйңҖиҰҒжҸҗдҫӣеӮіе…Ҙ variable
  • жҜҸеҖӢ module йғҪжңғйңҖиҰҒжҸҗдҫӣ provider

Terraform зҡ„иӘһжі•иҰҒжұӮдёҠйқўйҖҷдәӣеҸғж•ёйғҪжҳҺзўәзҡ„е®ҡзҫ©пјҢйҖҷи®“ж•ҙеҖӢ module жҲ–иіҮж–ҷеӨҫзҡ„жҸҸиҝ°йқһеёёжё…жҘҡгҖӮдҪҶжңғиҷ•е°ұжҳҜпјҢйҖҷдәӣжҸҸиҝ°зҡ„еҸғж•ёйҒ“еҮәйғҪжҳҜпјҢиҖҢдё”дёҚж–·зҡ„йҮҚиӨҮпјҢжҜҸеҖӢ module жҲ–иіҮж–ҷеӨҫйғҪиҰҒжҸҗдҫӣпјҢдёҚз„¶еңЁ terraform validate жҷӮжңғеӣ зӮәжҮүжҸҗдҫӣеҸғж•ёжңӘжҸҗдҫӣпјҢе°ҺиҮҙйҢҜиӘӨгҖӮйӣ–з„¶з«Ӣж„ҸиүҜеҘҪпјҢдҪҶеҚ»еҡҙйҮҚйҒ•еҸҚ DRY (Don’t Repeat Yourself) зҡ„еҺҹеүҮ

  • module еӨҡеұӨеј•з”ЁпјҢproject еј•з”ЁеӨ§ moduleпјҢеӨ§ module еҸҲеј•з”Ёе°Ҹ module
  • й–Ӣе§Ӣж„ҹиҰә backend иҲҮ provider зҡ„ code жҜ”е…¶д»–еҠҹиғҪ code еӨҡдәҶ

жңүжІ’жңүеҸҜиғҪз”Ёе…¶д»–е·Ҙе…·пјҢйҒҝе…ҚйҖҷдәӣйҮҚиӨҮзҡ„еҸғж•ёпјҢbackendпјҢprovider зӯүзӯүпјҹ

Terragrunt#

жҺЁи–ҰжңүеӨ§йҮҸдҪҝз”Ёзҡ„еңҳйҡҠпјҢзӣҙжҺҘдҪҝз”Ё Terragrunt йҖҷж¬ҫе·Ҙе…·гҖӮ

д»–жҳҜдёҖеұӨ terraform зҡ„ wrapper

  • i.e. еҹ·иЎҢ terraform plan -> terragrunt plan
  • terragrunt жңғе…Ҳи§ЈжһҗпјҢз”ўз”ҹйҮҚиӨҮзҡ„ codeпјҢз„¶еҫҢеҶҚеҹ·иЎҢ terraform
  • з·ЁијҜжҷӮеҸӘиҰҒеҜ«дёҖж¬ЎпјҢterragrunt д»ЈзӮәеңЁе…¶д»–ең°ж–№з”ўз”ҹйҮҚиӨҮзҡ„ code
  • з”ўз”ҹзҡ„ code жңғиў«йҡұи—ҸиҲҮ cache

Terragrunt жңүиЁұеӨҡеҠҹиғҪпјҢдҫӢеҰӮ

  • иҷ•зҗҶ BackendпјҢproviderпјҢиҲҮе…¶д»–дёҚж–·йҮҚиӨҮзҡ„ tf code
  • иҷ•зҗҶеӨҡз’°еўғдёӢйҮҚиӨҮзҡ„ tf
  • …е®Ңж•ҙеҠҹиғҪи«ӢиҰӢе®ҳж–№ж–Үд»¶

е°Һе…Ҙ Terragrunt еҸҜд»ҘйҒҝе…ҚйҮҚиӨҮд»ЈзўјпјҢйҷҚдҪҺз¶ӯиӯ·жҲҗжң¬пјҢжҸҗеҚҮз”ҹз”ўж•ҲзҺҮ

tf жӘ”жЎҲпјҢеңЁиЁұеӨҡ block дёӯзҰҒжӯўдҪҝз”Ё variableпјҢйҖҷеұӨйҷҗеҲ¶жңүе…¶иҖғйҮҸпјҢдҪҶжҳҜзјәй»һжҳҜйҖ жҲҗиЁұеӨҡ hard coded зҡ„зЁӢејҸзўјгҖӮ

Terragrunt з”ўз”ҹзҡ„зЁӢејҸзўјпјҢз”ұж–јжҳҜеңЁ terragrunt еҹ·иЎҢеҫҢпјҢterraform еҹ·иЎҢеүҚпјҢеӣ ж¬ЎжІ’жңүйҖҷеұӨйҷҗеҲ¶пјҢеҸҜд»ҘеҒҡиЁұеӨҡдәӢжғ…пјҢдҫӢеҰӮ code generate д»ҘеҸҠ function йҒӢз®—пјҢterragrunt жҸҗдҫӣиЁұеӨҡе…§е»ә functionпјҢйҖҷйӮҠеҸӘд»Ӣзҙ№еёёз”Ёе№ҫеҖӢгҖӮ

  • find_in_parent_folder()
  • path_relative_to_include()

зҜ„дҫӢе…©еҖӢжӘ”жЎҲ

project/dev/terraform.hcl

include {
  path = find_in_parent_folders()
}

project/terraform.hcl

env = path_relative_to_include()

жңҖеҫҢ parse еҫҢзҡ„зөҗжһң

project/dev/terraform.hcl

env = "dev"

еҒҡдәҶе…©д»¶дәӢ

  • иҰҒжұӮ project/dev еҺ»дёҠеұӨе°Ӣжүҫ terraform.hclпјҢдёҰдё”еј•е…Ҙе…¶дёӯзҡ„иЁӯе®ҡ (env="")
    • еӯҗе°ҲжЎҲеҸҜд»Ҙ include жҜҚиіҮж–ҷеӨҫзҡ„зЁӢејҸзўј
  • еҢҜе…ҘдёҠеұӨ project жҷӮпјҢеҜ«е…Ҙзӣёе°Қзҡ„и·Ҝеҫ‘ (dev)пјҢдёҰдё”её¶е…Ҙ (env=“dev”)
    • жҜҚе°ҲжЎҲеҸҜд»ҘзӮәеҗ„еҖӢеӯҗзӣ®йҢ„й…ҚзҪ®зӣёе°Қи·Ҝеҫ‘
