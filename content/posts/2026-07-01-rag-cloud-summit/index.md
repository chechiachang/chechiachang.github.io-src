## Workshop

從零開始建立自己的第一個 RAG 系統

### Outline

RAG（Retrieval-Augmented Generation）是一種結合檢索與生成的技術，能夠提升 AI 模型在特定領域的知識表現。本工作坊將帶領學員從零開始，實作一個基於 RAG 的智能問答系統，結合 OpenAI 的語言模型、LangChain 的工具鏈以及 Qdrant 向量資料庫，打造一個高效且準確的知識搜尋與分享平台。

流程
- 釐清什麼情境需要導入 RAG（Retrieval-Augmented Generation）
- 理解 Embedding 與向量資料庫的核心機制
- 實際建置 Embedding Search 並驗證查詢效果
- 透過上機練習鞏固流程
- 將 OpenAI 與 Qdrant 接上，打造自動化問答體驗
- 評估模型表現並進行驗證

# Target group

工作坊結束後，學員將能夠：
- 理解並實作完整的 RAG 流程，將現有文檔轉為可檢索的知識庫
- 使用 Python 與 OpenAI 建構 Retrieval-Based QA
- 利用 Qdrant 向量資料庫提升知識搜尋的速度與準確性
- 透過理論與實作並重的設計，確保每位學員都能實際操作

本次工作坊使用自己的電腦，實際部署並操作 RAG AI Agent。請自備筆電，透過 docker 與 jupyter notebook 完成練習。
